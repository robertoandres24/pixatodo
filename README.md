# pixatodo
a vuejs chrome extension

### Link to my Chrome Web Extension
https://chrome.google.com/webstore/detail/pixatodo/bdgdpelmmdancnfaakbekmapffolbnij?hl=es-419   

### Development mode
`npm run watch:dev`
### Build a new version
change the version in **manifest.json**   
`npm run build`   
`npm run build-zip`
